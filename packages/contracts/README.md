# @kiri_ikki/thread-contracts

Zod schemas and TypeScript types for the self-hosted CopilotKit Thread
Platform. Most applications should install `@kiri_ikki/thread-react` instead;
use this package directly when validating server responses or sharing thread
types outside React.

See the [Consumer Quickstart](https://github.com/Hieuej147/Copilotkit-threads/blob/main/docs/CONSUMER_QUICKSTART.md).
